# C-Projects

## Overview
This repository contains a collection of C programming projects developed to strengthen my understanding of the C programming language. These projects cover fundamental programming concepts and demonstrate problem-solving skills through practical implementations.

## Concepts Covered
- Variables and Data Types
- Operators
- Conditional Statements (if, if-else, switch)
- Loops (for, while, do-while)
- Functions
- Arrays
- Strings
- Pointers
- Structures
- File Handling
- Dynamic Memory Allocation
- Menu-Driven Programs

## Project Structure
Each project is organized into its own folder and contains the source code along with a brief description.

## Requirements
-GCC Compiler
-Linux or Windows
-Command Line/Terminal

## Compilation
Compile any C program using:
gcc filename.c

## Execution
Run the compiled program using:
./a.out
