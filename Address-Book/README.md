## Overview
The Address Book is a C programming project designed to store and manage contact information efficiently. It allows users to maintain a digital contact list by performing various operations through a simple menu-driven interface.

## Objective
- Create new contacts and store them in a file.
- Search for existing contacts using different search criteria.
- Edit or update contact details whenever required.
- Delete specific contacts from the address book.
- Display the complete list of saved contacts.
- Save contact information to a file and retrieve it whenever the application is executed.

## Features
- Add new contacts
- Search contacts
- Edit contact details
- Delete contacts
- Display all contacts
- Persistent storage using file handling

## Technologies Used
- C Programming
- File Handling
- Functions
- Structures
- String Manipulation

## Compilation
All the files can be compiled using: gcc *.c

## Execution
The compiled files can be executed using: ./a.out
